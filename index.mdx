Point Cloud Library (PCL) is a standalone, large-scale open project for 2D/3D image and point cloud processing. PCL contains state-of-the-art algorithms for filtering, feature estimation, surface reconstruction, registration, model fitting, segmentation, and visualization — all implemented in C++17.

## Get started in four steps

<Steps>
  <Step title="Install PCL">
    Install via your system package manager or build from source. PCL is available on Ubuntu, macOS (Homebrew), and Windows (vcpkg).

    ```bash
    # Ubuntu
    sudo apt install libpcl-dev

    # macOS
    brew install pcl
    ```
  </Step>
  <Step title="Set up your CMake project">
    Add PCL as a dependency in your `CMakeLists.txt` and link against the modules you need.

    ```cmake CMakeLists.txt
    find_package(PCL REQUIRED COMPONENTS common io filters)
    target_link_libraries(my_app ${PCL_LIBRARIES})
    ```
  </Step>
  <Step title="Load and process a point cloud">
    Use `pcl::PCDReader` to load a `.pcd` file and apply a filter in just a few lines.

    ```cpp main.cpp
    #include <pcl/io/pcd_io.h>
    #include <pcl/point_types.h>
    #include <pcl/filters/voxel_grid.h>

    int main() {
      pcl::PointCloud<pcl::PointXYZ>::Ptr cloud(new pcl::PointCloud<pcl::PointXYZ>);
      pcl::io::loadPCDFile("input.pcd", *cloud);

      pcl::VoxelGrid<pcl::PointXYZ> vg;
      vg.setInputCloud(cloud);
      vg.setLeafSize(0.01f, 0.01f, 0.01f);

      pcl::PointCloud<pcl::PointXYZ>::Ptr filtered(new pcl::PointCloud<pcl::PointXYZ>);
      vg.filter(*filtered);
      return 0;
    }
    ```
  </Step>
  <Step title="Explore the modules">
    PCL is organized into focused modules. Browse the [Module Reference](/modules/filters) to discover algorithms for your use case.
  </Step>
</Steps>
